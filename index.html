Nice To Meet You

Your university college events organizer

Project vision

Imagine you are in a university residence, where you can meet a great amount of people. NTMY, giving you the opportunity to organise thematic events, will help you to meet new friends!

Every user wears a coloured bracelet. With the app you can create a new meeting/event in the college to know new people (like see a football match in tv or do a card game). The system will then set a specific location for the event (considering the type of the event and the free college’s place) and it will assign it a colour; the user’s bracelet will turn that colour too (e.g. blue). Whenever a new event is created, people in the college with the same interests get a notification and, if they decide to participate at the blue event, their bracelet will become blue. The lights in the reserved room will also become blue to indicate that the room will host the blue event, and in the main streets of the college some indicators will color of blue, to easily find the place.

The system also acts according to the category of the meeting (e.g. event with TV, board game event, generic discussion event, music event/dance, sporting event), adjusting the lights, the video (tv) and the audio.

Then, when the user is at the event and greet someone else, the system will detect (via your bracelet) the interaction and register his personal information (such as name, photo, phone number, Facebook/Instagram, ...) on the app, so that at the end of the day the user can always remember who he met. No more forgotten names or time lost in exchanging phone numbers!

Ambient Intelligence main steps

Sensing

Each user has got a coloured bracelet, which is capable of registering all the new people that the user has met at the event by transmitting and receiving some data (such as an ID code) from/to the nearby devices. The input for starting the communication is given by the handshake between two users.

Reasoning

When an user creates a new event/meeting, the system chooses the most suitable place available to host it according to the kind of event, and assigns it a colour. According to the category of the event, the system is also able to determine how to correctly set the ambient characteristics such as lights, video and audio.

Acting

Once the system has chosen the event's colour, it will adjust some lights in the selected place and some markers around the college to match that colour. It can also set the TV program, the audio and the right illumination based on the kind of event.

Interacting

The user can interact with the system with a mobile app, which lets the user create new events, choose what info to share with the others and check the list of the new contacts he met at the events, with all the info they shared.

Ambient Intelligence features

NTMY is...

...sensitive, because it will detect when you are shaking someone's hand.

...responsive, because it will suggest you the events that fit with your interests.

...adaptive, because it will adapt the environment according to the scheduled event.

...transparent, because the interaction is as natural as a handshake.

...ubiquitous, because you can interact with it everywhere in order to find the way to your event.

...intelligent, because it will smartly allocate the available rooms according to the types of events.

System architecture

Hardware Architecture

  • PC running the Python Flask app
  • Used to store and retrieve users and events data
  • A Raspberry client interfaced with the other devices in the room
  • Vibration on hanshake recognition
  • Lights up of the color of the event
  • Bluetooth to pair with the phone
  • Managed by a Raspberry board
  • A circuit with Led lights that will form an arrow pointing the direction
  • Nfc reader that will read user's NFC tag
  • Philips Hue lights
  • Smart TV
  • Stereo system

Software Architecture

  • Smartly assigns rooms to each event
  • Calculate the route from user's room and the one of the event
  • Generate movie and music playlists according to the users partecipating in that event
  • Adjusts lights, TV and audio to fit the category of the event
  • Adapts ambient characteristics dinamically
  • Made for Android devices
  • Events management functions (create/join)
  • Shows information about the people you met
  • Recognizes the handshake and starts the data sharing
  • Lights up with the event's color
